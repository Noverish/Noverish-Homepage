고려대학교 먹거리 플랫폼

CTO
iOS & Server Developer

고려대학교의 모든 식당, 카페, 술집의 정보를 제공하고 사용자들의 후기를 공유하는 먹거리 플랫폼입니다.
이 서비스에서 CTO를 역임하였으며, iOS와 Server 개발자로써 이 서비스를 개발했습니다.

Amazon Web Service - API Gateway & Lambda 사용

이 두 서비스를 사용해서 서버리스 백엔드를 구축했습니다.
Nodejs를 이용해서 Lambda function의 코드를 구현했습니다.
또한 요즘 추세에 맞게 Server API는 RESTful하게 구현했습니다.

ApiDoc - API 문서화 도구 사용

다른 프론트엔드 개발자가 우리의 백엔드 서비스를 쉽게 사용하기 위해서 ApiDoc을 이용해서 API를 문서화 했습니다.

Google Firebase Crash Reporting 사용

사용자의 기기에서 일어난 모든 Unhandled Exception을 통합 관리 및 추적을 할 수 있는 서비스입니다.
이 서비스를 사용해서 특정 기기, 특정 상황에서만 일어날 수 있는 Exception들을 수정했습니다.

Google Analytics 사용

사용자 분석 서비스로 현재 이용하고 있는 사용자수, 화면 위치, 기기나 지역든 모든 정보를 한 눈에 볼 수 있습니다.
이 서비스를 사용하기 위한 코드 및 SDK를 어플리케이션에 넣었으며 이 정보들을 조합 및 분석해서 마케팅에 활용했습니다.
